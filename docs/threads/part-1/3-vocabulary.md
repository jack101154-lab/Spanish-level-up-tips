# 词汇篇

西语词汇学习不能只背“中文意思”。你要同时记住性别、复数、搭配、常用场景和一个能直接开口的例句。

## 本章速览

- 名词必须连冠词一起记：el problema, la mano。
- 动词要按场景记，不要孤立背原形。
- 优先掌握高频功能词：que, de, a, en, con, para, por。
- 每个词至少绑定一个句子。

## 名词：冠词一起记

不要只记 casa = 房子。要记：

- la casa
- una casa grande
- Vivo en una casa pequeña.

原因很简单：西语里名词性别会影响冠词、形容词和代词。你一开始偷懒，后面每句话都会返工。

常见例外也要早见面：

- el problema
- el mapa
- la mano
- la foto

## 动词：按“我能说什么”来记

背 hablar = 说话，不如直接背：

- hablo español
- hablo un poco de español
- quiero hablar contigo
- no entiendo, ¿puedes repetir?

动词是西语句子的发动机。初学阶段优先掌握这些高频动词：

- ser, estar, tener, haber
- ir, venir, hacer, decir
- querer, poder, deber, necesitar
- hablar, comer, vivir, estudiar, trabajar

## 高频小词非常重要

很多句子的难点不是名词，而是小词：

- de：of/from/about
- a：to/at
- en：in/on/at
- con：with
- para：for/in order to
- por：because of/through/by
- que：that/what/than

这些词需要通过大量例句吸收，不能只背一个中文释义。

## 词汇卡片模板

每个词条建议记录：

- 单词：trabajo
- 词性：名词，阳性
- 搭配：buscar trabajo, tener trabajo
- 例句：Estoy buscando trabajo.
- 变体：trabajar, trabajador
- 场景：求职、日常介绍

## 每日词汇训练

每天 20 到 30 分钟即可：

1. 新学 10 个词，不求多。
2. 每个词写一个自己的句子。
3. 听一遍发音。
4. 第二天复习并改写句子。
5. 周末用本周词汇写一篇短文。
