# 内容质量检查清单

这个清单用于每次新增或修改章节后自查。

## 西语准确性

- 名词性别是否正确？
- 形容词是否和名词性数一致？
- 动词变位是否匹配主语？
- 重音符号是否正确？
- 问号和感叹号是否成对使用？
- 例句是否自然，不是中文直译？

## 学习有效性

- 本章是否解决了一个明确学习问题？
- 是否有“为什么”和“怎么练”？
- 是否给了可执行练习？
- 是否避免一次讲太多概念？
- 是否提醒学习者输出和复盘？

## 对中文母语者友好

- 是否解释了中文学习者容易误解的地方？
- 是否避免只用英文术语堆叠？
- 是否提供了中文理解路径？
- 是否指出常见错误？

## 项目一致性

- 标题层级是否清晰？
- 文件名是否使用小写英文和连字符？
- SUMMARY 是否更新？
- 链接是否相对路径正确？
- 表格是否在移动端也可读？

## 版权与引用

- 是否避免大段复制教材、课程、文章？
- 引用外部资源时是否说明来源？
- 是否优先使用官方、开放或长期稳定资源？
- 是否避免把 AI 生成内容伪装成权威结论？
