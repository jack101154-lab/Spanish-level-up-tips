# 其他语种推荐

如果继续沿用这个 GitBook 文档型学习框架，最适合做的语种通常有三个特点：使用资源丰富、能力等级清晰、学习路径可以拆成听说读写模块。

## 最推荐

### 法语

优点：CEFR 和考试体系成熟，中文学习者需求稳定，文化、留学、移民、工作场景都能覆盖。

需要加强：发音、连读、阴阳性、动词变位、听力弱化。

### 意大利语

优点：拼读友好，入门体验好，音乐、艺术、旅行、美食主题天然适合做内容。

需要加强：动词变位、冠词、介词和口语场景。

### 葡萄牙语

优点：巴西葡语有大量音乐、影视和生活素材，适合做兴趣驱动项目。

需要加强：发音、鼻化元音、欧洲葡语与巴西葡语差异。

## 可以做，但要改框架

### 德语

可以做成很专业的项目，但不能完全照搬英语框架。需要增加格、词序、冠词变化、可分动词、复合词等章节。

### 日语和韩语

也值得做，但需要重构项目。它们必须从文字系统、敬语、语序、助词或语尾开始设计，不适合直接沿用这套西语/英语式目录。

## 我的建议

如果你的目标是做一个成熟、容易扩展、适合中文学习者的开源项目，优先级是：

1. 西班牙语
2. 法语
3. 德语
4. 日语

西班牙语综合性最好：受众广、入门门槛适中、材料多、体系成熟，也最适合先做出一个能跑通的样板。
