# Common

这不是完整词典，而是初学阶段最值得优先掌握的高频词和表达。建议把每个词放进句子里练。

## 生存表达

| 西语 | 中文 |
| --- | --- |
| Hola | 你好 |
| Buenos días | 早上好 |
| Buenas tardes | 下午好 |
| Buenas noches | 晚上好/晚安 |
| Gracias | 谢谢 |
| De nada | 不客气 |
| Por favor | 请 |
| Perdón | 抱歉/打扰一下 |
| No entiendo | 我不明白 |
| ¿Puedes repetir? | 你可以重复一遍吗 |
| ¿Cuánto cuesta? | 多少钱 |
| ¿Dónde está...? | 在哪里 |

## 高频动词

| 原形 | 常用句 | 中文 |
| --- | --- | --- |
| ser | Soy estudiante. | 我是学生 |
| estar | Estoy en casa. | 我在家 |
| tener | Tengo tiempo. | 我有时间 |
| ir | Voy al trabajo. | 我去工作 |
| hacer | Hago ejercicio. | 我运动 |
| querer | Quiero café. | 我想要咖啡 |
| poder | Puedo ayudarte. | 我可以帮你 |
| necesitar | Necesito ayuda. | 我需要帮助 |
| hablar | Hablo español. | 我说西语 |
| aprender | Aprendo español. | 我学西语 |

## 高频连接词

| 西语 | 中文 | 例句 |
| --- | --- | --- |
| y | 和 | café y agua |
| pero | 但是 | Quiero ir, pero no puedo. |
| porque | 因为 | Estudio porque me gusta. |
| también | 也 | Yo también. |
| entonces | 那么/于是 | Entonces, vamos. |
| primero | 首先 | Primero estudio. |
| después | 然后 | Después trabajo. |
