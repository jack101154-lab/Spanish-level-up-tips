# Spanish Level-Up Tips

欢迎来到西班牙语学习指南。  
Welcome to the Spanish learning guide.

这份指南面向中文母语者和英语母语者。它的目标不是让你收藏一堆资源，而是帮助你建立一个能坚持、能复盘、能输出的西班牙语学习系统。

This guide is designed for Chinese and English speakers. Its goal is not to give you another pile of resources, but to help you build a Spanish learning system that is sustainable, reviewable, and output-driven.

建议阅读顺序 / Suggested Reading Order:

1. 先读 [认知篇 / Mindset](threads/part-1/1-understanding.md)，确认目标与路线。
2. 再读 [发音与拼读篇 / Pronunciation and Spelling](threads/part-1/2-pronunciation.md)，先把声音系统打稳。
3. 按 [12 周学习路线 / 12-Week Roadmap](threads/plans/12-week-roadmap.md) 执行。
4. 每周使用 [AI 学习篇 / Learning with AI](threads/part-1/8-ai.md) 的模板做口语、写作和纠错。

For English-speaking learners, the current project homepage is bilingual, while the full chapter content is being expanded progressively. Contributions that help create natural English explanations are very welcome.
