# Spanish Level-Up Tips

一份面向中文母语者与英语母语者的西班牙语进阶学习指南。  
An open-source Spanish learning guide for Chinese and English speakers.

## 项目介绍 / Project Overview

欢迎来到 **Spanish Level-Up Tips**。

这是一个为中文母语者和英语母语者设计的西班牙语学习开源项目。我们希望把零散的学习经验、教材笔记、考试技巧、资源推荐和 AI 学习方法，整理成一份结构清晰、长期可维护、真正能帮助学习者行动起来的西班牙语进阶指南。

西班牙语并不是一门只靠背单词就能学好的语言。它有非常友好的拼读规则，也有让初学者头疼的动词变位、阴阳性、ser/estar、宾格代词、前置词和不同地区表达差异。这个项目的目标，就是把这些难点拆开、讲清楚，并配上可执行的训练方法。

我们不想做一份“收藏夹式”的资源堆砌，而是希望做成一套完整的学习系统：

- 初学者知道从哪里开始；
- 自学者知道每天练什么；
- 备考者知道如何对齐 DELE/SIELE 和 CEFR 等级；
- 进阶学习者能找到听说读写、地区差异和真实材料的训练路径；
- 贡献者可以持续补充更好的例句、练习、资源和学习经验。

如果你也在学西班牙语，或者你已经走过某个阶段，欢迎一起完善这个项目。哪怕只是修正一个例句、补充一个资源、分享一个学习坑点，都会让后来的人少走一点弯路。

Welcome to **Spanish Level-Up Tips**.

This is an open-source Spanish learning project for both Chinese and English speakers. It turns scattered learning notes, textbook insights, exam strategies, resource lists, and AI-assisted workflows into a structured, maintainable guide that learners can actually follow.

Spanish is not a language you master by memorizing word lists alone. It has friendly spelling and pronunciation rules, but learners often struggle with verb conjugation, noun gender, adjective agreement, ser/estar, object pronouns, prepositions, and regional variation. This project aims to explain those challenges clearly and connect them with practical training routines.

We do not want this to become a random bookmark collection. The goal is to build a complete learning system:

- beginners know where to start;
- self-learners know what to practice every day;
- exam learners can align their study with CEFR, DELE, and SIELE;
- intermediate learners can train listening, speaking, reading, writing, and regional usage;
- contributors can keep improving examples, exercises, resources, and learning stories.

If you are learning Spanish, teaching Spanish, or have already passed through a certain stage, you are warmly invited to improve this project with us.

## 适合谁 / Who This Is For

- 零基础到 A2，希望建立稳定学习系统的人
- 学过一点西语，但卡在动词变位和开口表达的人
- 想用西语旅行、社交、工作、读新闻或看影视的人
- 准备 DELE A2/B1/B2 或 SIELE 的学习者
- 希望用 AI 做长期陪练，而不是偶尔翻译几句话的人

- Absolute beginners through A2 learners who want a stable study system
- Learners who know some Spanish but get stuck on conjugation or speaking
- People who want to use Spanish for travel, work, culture, reading, or media
- DELE A2/B1/B2 or SIELE learners
- Learners who want to use AI as a long-term coach, not just a translator

## 学习原则 / Learning Principles

1. 先把声音学准，再大量输入。
2. 动词变位不靠死背全部表格，而靠高频句型和场景重复。
3. 语法服务表达，不把语法当成孤立知识点。
4. 从第一周开始开口，哪怕只说三句话。
5. 每周固定产出：一段录音、一篇短文、一份错题/错句清单。

1. Build a clear sound model before increasing input.
2. Learn conjugation through high-frequency patterns and real contexts, not only tables.
3. Treat grammar as a tool for expression, not isolated knowledge.
4. Start speaking in week one, even if you can only say three sentences.
5. Produce something every week: one recording, one short text, and one error log.

We welcome careful, friendly, reusable contributions, including:

- adding listening, reading, speaking, and writing exercises;

We especially welcome explanations that help learners understand why something is difficult, how to practice it, and how it is used in real situations.

## 为什么西班牙语适合这个框架

西班牙语非常适合做成这种文档型学习指南：它使用拉丁字母，拼读规则稳定，材料资源丰富，CEFR、DELE、SIELE 的能力等级也很成熟。中文学习者只要把「声音、变位、性数一致、真实输入」这四件事打通，进步会很明显。

如果想用同一框架做其他语种，最推荐：

- 法语：同属罗曼语，适合沿用听说读写和 CEFR 框架，但要加强发音、连读和阴阳性。
- 意大利语：拼读友好，音乐、旅行、文化素材多，适合初学者做兴趣型项目。
- 葡萄牙语：适合做巴西葡语方向，口语和音乐素材丰富，但发音体系要单独加强。
- 德语：适合做严肃进阶项目，但需要增加格、词序、可分动词等独立章节。

不太建议直接套用这个框架的语种：日语、韩语、阿拉伯语、俄语。它们需要单独处理文字系统、敬语/格/书写系统等问题，项目结构应重新设计。
