# Contributing

欢迎一起完善 Spanish Level-Up Tips。

这个项目希望保持四个特点：对中文和英语学习者友好、方法可执行、内容能长期维护、例句和解释尽量准确自然。

## 可以贡献什么

- 修正西班牙语例句、语法解释或中文表述。
- 补充英文解释，让英语母语者也能顺畅阅读。
- 补充 A1-B2 分级练习。
- 增加听力、阅读、词汇、考试资源。
- 增加拉美不同国家与西班牙本土的用法差异。
- 补充 DELE/SIELE 备考经验。
- 改进排版、目录、图表和学习路径。

## 写作原则

1. 先讲清楚学习者要解决的问题，再讲规则。
2. 每个语法点尽量配 3 到 5 个例句。
3. 不堆资源，优先解释资源适合什么水平、怎么用。
4. 尽量区分“通用西语”和“地区用法”。
5. 不制造焦虑，强调可持续练习。
6. 如果新增英文内容，优先写自然英文，不要机械逐字翻译中文。

## 章节格式

建议每章使用这个结构：

```markdown
# 章节标题

## 本章速览

- 关键结论 1
- 关键结论 2

## 为什么重要

解释学习者常见误区。

## 怎么练

给出具体步骤。

## 练习

给出可执行任务。
```

## 内容质量检查

提交前请确认：

- 西语例句拼写、重音符号、标点基本正确。
- 中文或英文解释不把复杂问题说得过度绝对。
- 链接可访问，且尽量指向官方或长期稳定来源。
- 没有直接复制教材、课程或付费资料的大段内容。
- 如果涉及 DELE/SIELE、CEFR 或报名政策，请优先核对官方信息。
